Glossário Técnico & Padrões Geográficos

Central de Conhecimento

Desmistificando os termos técnicos de geoprocessamento e sistemas de informação geográfica para conectar arquitetos, engenheiros e gestores à fonte dos dados.

Informação urbanística direto da fonte
WMS

Web Map Service: Padrão OGC que provê imagens georreferenciadas (raster) através da internet. Ideal para visualização em plano de fundo no QGIS ou ArcGIS sem a necessidade de baixar arquivos pesados.

WFS

Web Feature Service: Permite a recuperação e manipulação direta de feições vetoriais (pontos, linhas e polígonos) com seus atributos, garantindo acesso completo aos parâmetros numéricos e tabelas de zoneamento.

GeoJSON

Formato aberto e leve estruturado em JSON para representação de geometria espacial e dados não espaciais. É o padrão nativo para aplicações web modernas e bibliotecas como Leaflet e Mapbox.

ArcGIS REST

Interface de programação (API) mantida pelos ecossistemas Esri que permite consultar mapas, camadas de atributos e serviços de geocodificação disponibilizados por prefeituras.

GeoServer

Servidor open-source em Java utilizado por administrações públicas para compartilhar geodados e servir camadas sob os padrões abertos OGC (WMS, WFS, WCS).

SIG

Sistema de Informação Geográfica: Conjunto integrado de hardware, software e dados espaciais projetado para capturar, armazenar, manipular e analisar informações sobre o território.
